# Quiz App / Test Platform

A web application built with React and Vite designed to help developers test their knowledge in various programming languages including Python, JavaScript, and SQL.

## Features

- **Subject-Specific Quizzes**: Challenge yourself with tailored questions in Python, JavaScript, and SQL.
- **Responsive Design**: A modern, responsive interface that works seamlessly on desktop and mobile devices.
- **Interactive UI**: Built with React and tailored with Tailwind CSS for a dynamic user experience.
- **React Router Navigation**: Smooth client-side routing for easy navigation between different sections.

## Tech Stack

- **[React](https://react.dev/)**: JavaScript library for building user interfaces.
- **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework for rapid UI development.
- **[React Router](https://reactrouter.com/)**: Declarative routing for React web applications.

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Make sure you have Node.js and npm installed.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd Test-Platform
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to the local server address (usually `http://localhost:5173`).
